# Client Onboarding Automation

An n8n workflow that automates the entire client onboarding process.

## What it does
- Creates a dedicated Google Drive folder for the new client
- Sends a personalized welcome email via Gmail
- Alerts the team instantly via Telegram

## Tools Used
- n8n
- Google Drive API
- Gmail API
- Telegram Bot API

## How to use
1. Download the workflow JSON file
2. Import it into your n8n instance
3. Add your credentials (Google, Telegram)
4. Activate the workflow
